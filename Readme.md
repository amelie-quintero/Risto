# Risto

A general restaurant app written in Java Spring Boot. This project is planned to have the following features

- Creating and editing of Admin and User accounts with secure authentication.
- Users can place orders with the dishes that they want.
- Verification of out-of-stock ingredients triggering out-of-stock dishes when ordering.
- Ingredient inventory management with customizable low threshold for each ingredient.
- Cooks able to process orders and mark as completed.
- Recipe management with orders linked to ingredients.
